# Miya Portfolio Website

A starter personal portfolio website built with React and Vite.

## What is included

- Multi-page style navigation using hash routes
- Home, About, Projects, Writing, Activities, Resume, and Contact pages
- Clean starter layout with placeholders you can replace later
- Works well for GitHub upload and deployment on Vercel, Netlify, or GitHub Pages

## How to run locally

1. Install Node.js
2. Open the project folder in your terminal
3. Run:

```bash
npm install
npm run dev
```

## How to build

```bash
npm run build
```

The production files will appear in the `dist` folder.

## How to upload to GitHub

1. Create a new repository on GitHub
2. Upload all files from this project folder
3. Commit and push
4. Connect the repo to Vercel or Netlify for the easiest deployment

## Files to edit first

- `src/App.jsx` for your content
- `src/styles.css` for colors, spacing, and layout
- `index.html` for the page title and meta description

## Notes

This version avoids extra preview-only dependencies so it is easier to publish directly.
